Write a demo script for the current state of the app.

Rules:

- **2 minutes max.** Pitch judges are on the clock.
- **Open with the problem**, stated in one sentence that a Tunisian judge will feel viscerally.
- **Every claim must map to an on-screen action.** If you say "agents collaborate," the trace panel must visibly show agents. No talking about things that aren't running.
- **Include one emotional beat** — a testimonial quote, a number that shocks, or a visual the judge will remember.
- **Include fallback lines** for every stage direction, in case something breaks live (e.g. "and here you'd normally see the trace update — we have a recording just in case").
- **End with the ask.** What prize track, what next step, what you need from the sponsor. -**Make this a shark tank style pitch.** Judges should be leaning in, not checking their phones.

Format: a two-column table — Stage direction | Spoken line. Plus a 30-second backup version for the post-demo mingle.
